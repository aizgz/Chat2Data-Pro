# Chat2Data Pro

**Chat2Data Pro** 是一個輕量級的 AI 數據分析工具。無需複雜操作，直接上傳 Excel/CSV 表格，即可通過自然語言與數據對話。

---

## 🚀 快速開始

複製以下指令，在終端機一次執行即可完成配置：

```bash
# 1. 下載項目
git clone https://github.com/aizgz/Chat2Data-Pro.git
cd Chat2Data-Pro

# 2. 安裝依賴 (建議在虛擬環境中運行)
pip install -r requirements.txt

# 3. 啟動程序
python main.py

啟動成功後，請在瀏覽器訪問：http://127.0.0.1:8000
